# geditor

> Made with create-react-library

## Install

```bash
npm install --save geditor
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'geditor'
import 'geditor/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
